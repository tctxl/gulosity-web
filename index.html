Gulosity

Mysql Binlog插件，可直接集成至Spring

View on GitHub Download .zip Download .tar.gz

1.开启Mysql Binlog Replication.

GRANT REPLICATION SLAVE ON *.* to '账户名'@'主机' identified by '密码'

2.集成至项目

<dependency>
<groupId>com.opdar.gulosity</groupId>
<artifactId>gulosity-parse</artifactId>
<version>0.1</version>
</dependency>

3.Spring配置

4.实现Listener

实现RowCallback接口的监听器将获得到未处理过的原始数据，而继承自DefaultAutoMappingListener的监听器，将会自动匹配对应实体。

5.实现和表相对应的实体
